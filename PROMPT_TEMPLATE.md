# Ruby/Rails 学習用プロンプトテンプレート

新しいClaude Chatを開いて学習を始める際に、このプロンプトをコピー&ペーストしてください。

---

## 学習用プロンプト

```
Ruby/Railsの学習をサポートしてください。

【学習テーマ】
Day X: [ここにテーマ名を入力]

【学習項目】
- 項目1
- 項目2
- 項目3

【学習スタイル】
- 問題形式で実践的に学びたい
- コード例を交えて説明してほしい
- 「なぜそうするのか」を重視
- 良いコード/悪いコードの比較があると理解しやすい

【背景】
- エンジニア歴1年
- Railsでサービス開発経験あり（Next.js、Vercel Postgres使用）
- オブジェクト指向やデザインパターンを深く理解したい

【お願い】
1. 概念の説明
2. 実践的な課題を3つ程度出題
3. 各課題の解答例と解説
4. つまずきやすいポイントの説明

進め方を提案してください。
```

---

## 使い方

1. 上記プロンプトをコピー
2. 新しいClaude Chatを開く
3. 【学習テーマ】と【学習項目】を該当するIssueの内容に書き換えてペースト
4. 学習を進める
5. 終了後、Issueにコメント追加（Summary + Chat Link）してクローズ
